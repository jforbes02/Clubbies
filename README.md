Clubbies

A full-stack nightlife venue review and photo-sharing mobile application. Discover clubs, bars, and lounges, share photos, rate venues, and read reviews from the community.

Tech Stack:

  Backend:
  - FastAPI
  - PostgreSQL

  Frontend:
  - Flutter

Features

User Authentication – Register, login, JWT token refresh, secure logout
Venue Discovery – Browse venues with filters (type, capacity, price, age)
Ratings & Reviews – Rate venues 1-5 stars, write and manage reviews
Photo Sharing – Admin photo uploads for venues
User Profiles – View your rated venues and reviews
Search – Search venues by name/filters and users by username
